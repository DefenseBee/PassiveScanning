* **OWASP Nettacker**: [https://www.owasp.org/index.php/OWASP_Nettacker] - Automated penetration testing and automated Information Gathering.
* **GitRob:** [https://github.com/michenriksen/gitrob] - Gitrob will clone repositories belonging to a user or organization down to a configurable depth and iterate through the commit history and flag files that match signatures for potentially sensitive files. 
* **GitLeaks:** [https://github.com/zricethezav/gitleaks] - Gitleaks provides a way for you to find unencrypted secrets and other unwanted data types in git source code repositories.
